# ContactsManager

## C'est quoi ?
Un programme Java qui permet de gérer une liste de contacts.
On peut ajouter des contacts et les rechercher par leur nom.

## Comment lancer le programme ?
1. Ouvrir le projet dans IntelliJ
2. Lancer le fichier `ContactsManager.java`

## Ce que fait le programme
- Crée 5 contacts avec un nom et un numéro de téléphone
- Les ajoute dans le gestionnaire
- Recherche un contact par son nom et affiche son numéro
